# proyecto-monitorizacion-equipo-4
En este proyecto monitorearemos un Ubuntu Server 22.04 mediante Prometheus y diferentes exporters configurados por los administradores del servicio: Node Exporter (Sufian), NGINX Exporter (Oscar), Blackbox Exporter (Leo), HAProxy Exporter (Jhon) y MySQL Exporter (Rubén). Mediante el uso de estos exporters se supervisan métricas del sistema, servicios web y bases de datos en tiempo real.
